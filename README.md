# 黄金投资新闻雷达

一个免费、可公网访问的黄金投资新闻影响看板。页面每天自动更新，展示可能影响国际黄金和国内黄金价格的新闻，并按优先级、影响方向和影响范围整理。

## 本地使用

```bash
npm run check
```

`check` 会试抓取新闻并打印结果，但不会写入数据文件。

```bash
npm run update
```

`update` 会生成：

- `data/daily.json`
- `data/history.json`
- `data/archive/YYYY-MM-DD.json`

直接打开 `index.html`，或用任意静态服务器预览页面。

## 免费部署

推荐使用 GitHub Pages：

1. 创建 GitHub 仓库并推送本项目。
2. 在仓库设置里启用 GitHub Pages，来源选择 `GitHub Actions`。
3. `.github/workflows/daily-update.yml` 会每天 UTC 23:00 运行，对应北京时间约早上 7 点。
4. 工作流会抓取公开新闻源、生成数据、提交更新，并部署页面。

## 影响判断规则

初版不接入付费 AI 或付费新闻 API，使用规则评分：

- 降息、宽松、通胀、避险、地缘冲突：偏利多黄金。
- 加息、鹰派、美元走强、收益率上升：偏利空黄金。
- 央行购金、黄金储备、人民币汇率、上海金溢价：影响国内黄金，视关键词给出方向。

## 默认新闻源

- 美联储官方 RSS：新闻稿与货币政策更新。
- BLS 官方 RSS：CPI、就业、工资等美国宏观数据更新。
- Google News RSS 搜索：补充国际黄金与国内黄金相关新闻聚合。

## 免责声明

本页面仅用于新闻整理和投资研究参考，不构成投资建议。黄金价格受多重因素影响，请结合行情、仓位与风险承受能力独立判断。
